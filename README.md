# R for Data Science Community Project

This repository contains materials for a workplace learning initiative to introduce R programming and data science fundamentals.

## Structure

-   `curriculum/`: Weekly learning schedule
-   `announcements/`: Kickoff email and communications
-   `slides/`: Weekly slide outlines
-   `handouts/`: Printable handouts with exercises
-   `datasets/`: Sample data for practice

## Getting Started

1.  Clone this repository
2.  Open Positron IDE and start with Week 1 materials
3.  Follow the curriculum and complete exercises
